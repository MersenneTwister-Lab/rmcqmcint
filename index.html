Quasi Monte-Carlo Integration
with Low WAFOM Digital Net for R

This is an R packeage which calculates Quasi Monte-Carlo Integration using Quasi Monte-Calro Method. Low WAFOM Point Set is used for QMC.

Caution:Currently, this project is very experimental.
